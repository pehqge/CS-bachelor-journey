# INE5411 - Organização de Computadores I

Aqui estão os relatórios e códigos que fiz na disciplina. Ela cobre arquitetura de computadores, formatos de instrução e modos de endereçamento, programação em assembly, circuitos aritméticos e avaliação de desempenho. Todo o assembly foi escrito para MIPS e rodado no simulador MARS. A maior parte da disciplina foram os relatórios dos laboratórios, cada um com um ou dois exercícios.

## Relatórios dos laboratórios

| Relatório | Descrição |
| --- | --- |
| [Relatório 1](./Relatório%201) | Operações aritméticas em MIPS (A = B + 35 e C = D - A + E), com alocação das variáveis na memória. |
| [Relatório 2](./Relatório%202) | Exibição dos números de 0 a 9 em sequência num display de 7 segmentos. |
| [Relatório 3](./Relatório%203) | Multiplicação por somas sucessivas, feita dentro de um procedimento. |
| [Relatório 4](./Relatório%204) | Raiz quadrada pelo método de Newton, com um script Python antes da versão em assembly. |
| [Relatório 5](./Relatório%205) | Fatorial de n, com o valor de n lido pelo teclado. |
| [Relatório 6](./Relatório%206) | Média de dois vetores usando números de ponto flutuante. |
| [Relatório 7](./Relatório%207) | Exercícios com foco na alocação de dados, conferida pelo data segment. |
| [Relatório 8](./Relatório%208) | Operações com matriz de tamanho fixo. |

# INE5411 - Computer Organization I

These are the reports and code I did in the course. It covers computer architecture, instruction formats and addressing modes, assembly programming, arithmetic circuits and performance evaluation. All the assembly was written for MIPS and run on the MARS simulator. Most of the course was the lab reports, each with one or two exercises.

## Lab reports

| Report | Description |
| --- | --- |
| [Relatório 1](./Relatório%201) | Arithmetic in MIPS (A = B + 35 and C = D - A + E), with the variables allocated in memory. |
| [Relatório 2](./Relatório%202) | Showing the numbers 0 to 9 in sequence on a 7-segment display. |
| [Relatório 3](./Relatório%203) | Multiplication by successive additions, done inside a procedure. |
| [Relatório 4](./Relatório%204) | Square root by Newton's method, with a Python script before the assembly version. |
| [Relatório 5](./Relatório%205) | Factorial of n, with n read from the keyboard. |
| [Relatório 6](./Relatório%206) | Average of two vectors using floating-point numbers. |
| [Relatório 7](./Relatório%207) | Exercises focused on data allocation, checked through the data segment. |
| [Relatório 8](./Relatório%208) | Operations on a fixed-size matrix. |

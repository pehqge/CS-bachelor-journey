## Trabalho Prático 2

Monitorei três dispositivos (Android, notebook e roteador) no PRTG e capturei o tráfego no Wireshark para analisar os protocolos.

| Pasta/arquivo | O que é |
|---------------|---------|
| [PRTG](Trabalho%20Prático%202/PRTG) | Prints do monitoramento (Ping, Ping Jitter, HTTP, HTTP Advanced e Ethernet Traffic), separados por dispositivo. |
| [Wireshark](Trabalho%20Prático%202/Wireshark) | Capturas de ARP, HTTP e UDP, mais o arquivo `.pcapng`. |

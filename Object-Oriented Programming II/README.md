# INE5404 - Programação Orientada a Objetos II

Esta disciplina foi sobre orientação a objetos com Python. Passei por classes, encapsulamento, herança, classes abstratas, polimorfismo, composição, tratamento de erros, persistência em arquivo, interface gráfica e os princípios SOLID.

Fiz várias tarefas ao longo do semestre, umas sozinho e outras em grupo. As tarefas em grupo foram com o [Augusto Schweitzer](https://github.com/AugustoSchw), o [Rodrigo Schwartz](https://github.com/R0drigoSchwartz) e o [Tom Hunt](https://github.com/tmphnt).

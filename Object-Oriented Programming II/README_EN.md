# INE5404 - Object-Oriented Programming II

This course was about object orientation with Python. I went through classes, encapsulation, inheritance, abstract classes, polymorphism, composition, error handling, file persistence, graphical interfaces and the SOLID principles.

I did several assignments over the semester, some on my own and some in a group. The group work was done with [Augusto Schweitzer](https://github.com/AugustoSchw), [Rodrigo Schwartz](https://github.com/R0drigoSchwartz) and [Tom Hunt](https://github.com/tmphnt).

## Final Project: The Lost Key

The final project was an object-oriented game made as a group with me, [Augusto Schweitzer](https://github.com/AugustoSchw), [Rodrigo Schwartz](https://github.com/R0drigoSchwartz) and [Tom Hunt](https://github.com/tmphnt). The goal was to bring together everything we saw in the course: aggregation, composition, inheritance, polymorphism, clean code, SOLID and a UML diagram.

The finished game lives in its own repository, which here is the `Final Project` submodule. The [Trabalho Final](./Trabalho%20Final/) folder holds the supporting material.

Game repository: **[The Lost Key](https://github.com/tmphnt/The-Lost-Key)**

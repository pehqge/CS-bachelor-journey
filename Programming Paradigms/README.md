# INE5416 - Paradigmas de Programação

Aqui estão meus trabalhos da disciplina de Paradigmas de Programação, com foco nos paradigmas funcional e lógico. Os trabalhos maiores resolvem o quebra-cabeça Kojun em duas linguagens e um puzzle de lógica em Prolog. A pasta de exercícios tem práticas de Haskell do começo do curso.

## Trabalhos

| Trabalho | Linguagem / Paradigma | O que faz |
| :------- | :-------------------- | :-------- |
| [Trabalho 1](Trabalho%201) | Haskell (funcional) | Resolvedor do puzzle Kojun. O solver está em [solver.hs](Trabalho%201/solver.hs), junto com um scraper de problemas e testes automatizados. |
| [Trabalho 2](Trabalho%202) | Erlang (funcional) | O mesmo Kojun resolvido em Erlang, em [solver.erl](Trabalho%202/solver.erl). |
| [Trabalho 4](Trabalho%204) | Prolog (lógico) | Puzzle de lógica ao estilo do problema de Einstein, sobre astrônomos, em [astronomo.pl](Trabalho%204/astronomo.pl). |

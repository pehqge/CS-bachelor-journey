# INE5416 - Programming Paradigms

These are my assignments for the Programming Paradigms course, focused on the functional and logic paradigms. The larger assignments solve the Kojun puzzle in two languages and a logic puzzle in Prolog. The exercises folder holds Haskell practice from early in the course.

## Assignments

| Assignment | Language / Paradigm | What it does |
| :--------- | :------------------ | :----------- |
| [Trabalho 1](Trabalho%201) | Haskell (functional) | Kojun puzzle solver. The solver is in [solver.hs](Trabalho%201/solver.hs), together with a problem scraper and automated tests. |
| [Trabalho 2](Trabalho%202) | Erlang (functional) | The same Kojun solved in Erlang, in [solver.erl](Trabalho%202/solver.erl). |
| [Trabalho 4](Trabalho%204) | Prolog (logic) | An Einstein-style logic puzzle about astronomers, in [astronomo.pl](Trabalho%204/astronomo.pl). |

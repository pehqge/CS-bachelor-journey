# INE5418 - Computação Distribuída

## Sistema de Arquivos Distribuído (SMR)

**Grupo 17:** Gustavo Rodrigues Alves D'Angelo, Pedro Henrique Gimenez e Tom Pereira Hunt

Trabalho 2 da disciplina. Um sistema de arquivos distribuído fortemente consistente, construído sobre Replicação de Máquina de Estados (State Machine Replication - SMR), em Python puro. Suporta upload, download, listagem e remoção de arquivos, replicados entre múltiplos nós, mantendo consistência mesmo diante de falhas.

Detalhes de arquitetura, execução e demonstração estão no [README em inglês](README.md).

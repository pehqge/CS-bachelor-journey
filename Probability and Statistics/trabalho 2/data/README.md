Essa pasta contém os dois arquivos CSV tratados a partir da base pública do Metropolitan Museum of Art (MET), que foram utilizados nas análises deste trabalho.

- **`met_data.csv`**  
  Contém uma obra por linha (pintura ou escultura), com as tags listadas juntas em uma única célula, separadas por `|`. Ideal para análises gerais por objeto.

- **`met_data_tags.csv`**  
  Contém múltiplas linhas para cada obra, uma para cada tag associada. Ou seja, se uma obra tiver três temas, ela aparece três vezes — uma para cada tema. Esse formato é útil para análises de frequência e distribuição das tags.

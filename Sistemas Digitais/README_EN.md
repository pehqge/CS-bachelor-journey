# INE5406 - Digital Systems

This course continues what I saw in Digital Circuits, now with hardware design in VHDL on the FPGA board. The group work I did with Victória Rodrigues. The practical part has the in-class activities (in the [Aulas](./Aulas) folder) and four graded assignments over the semester. The first three revolve around the SAD (sum of absolute differences), in two versions (v1 and v3), and the last one is a convolution module with a Gaussian filter.

## Graded assignments

| Assignment | Description |
| --- | --- |
| [Prática 1](./Prática%201) | 4x1 mux, 1x4 demux and a BCD to 7-segment decoder in two forms (logic expressions and case). Comes with a report and the simulations. |
| [Prática 2](./Prática%202) | SAD v1 and v3 built from generic components (subtractor, adder, adder with overflow, mux and register). Report with the simulations and the delay analysis. |
| [Prática 3](./Prática%203) | SAD v1 and v3 with a testbench and golden model, using random stimuli generated in Python. |
| [Prática 4](./Prática%204) | Convolution module (Gaussian filter) in VHDL, with Python scripts to generate and filter the test matrices. Includes the PDF report and the RTL. |

# INE5406 - Sistemas Digitais

Disciplina que aprofunda os conhecimentos de Circuitos Digitais, com projeto de hardware em VHDL na placa FPGA. Trabalhos em grupo feitos com a Victória Rodrigues.

A parte prática é composta pelas atividades em aula e quatro práticas avaliativas ao longo do semestre, culminando na Prática IV — um **módulo de convolução (filtro gaussiano)** implementado em VHDL, com scripts Python para gerar e filtrar as matrizes de teste.

- `Atividades em Aula/` — exercícios feitos em aula
- `Prática 1` a `Prática 4` — as práticas avaliativas (VHDL + relatórios)
